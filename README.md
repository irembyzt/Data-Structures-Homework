# Sparse Matrix

# Tower of Hanoi

Bu program, Tower of Hanoi problemini recursive (özyinelemeli) fonksiyon
kullanarak çözer.
Kullanıcıdan disk sayısı alınır ve disklerin hangi çubuktan
hangi çubuğa taşındığı adım adım ekrana yazdırılır.

# Shunting Yard Algorithm

Bu program, Shunting Yard algoritması kullanarak
infix ifadeyi postfix ifadeye dönüştürür.
Stack veri yapısı kullanılmıştır.
Operatör önceliği ve parantezler dikkate alınmıştır.

gerçekleştirilmiştir.

- Min Heap: En küçük eleman kökte

# Double Linked List

Bu program, çift bağlı liste (Double Linked List) veri yapısını
C dili kullanarak uygular.

Özellikler:
- Başa ekleme
- Sona ekleme
- Düğüm silme
- Listeyi yazdırma

# Circular Linked List

Bu program, tek bağlı dairesel liste (Circular Linked List)
veri yapısını C dili ile uygular.

Özellikler:
- Başa ekleme
- Sona ekleme
- Listeyi yazdırma

# Array Insert & Delete

Bu program, dizi üzerinde belirli bir index'e
eleman ekleme ve silme işlemlerini gerçekleştirir.

Ekleme işleminde elemanlar sağa,
silme işleminde elemanlar sola kaydırılır.

# Graph Cluster (Connected Component)

BFS ise queue veri yapısı kullanılarak gerçekleştirilmiştir.
